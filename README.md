# Pokedex

Uma aplicação web simples para visualizar informações sobre Pokémon, utilizando a PokeAPI.

## Instalação
1. Clone o repositório: `git clone https://github.com/zzruanxx/Pokedex.git`
2. Abra o arquivo `index.html` em um navegador ou use um servidor local.

## Uso
- Abra `index.html` no navegador.
- Digite o nome ou ID de um Pokémon para buscar informações.

## Tecnologias
- HTML, CSS, JavaScript
- PokeAPI para dados dos Pokémon
